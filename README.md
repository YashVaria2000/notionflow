# NotionFlow

A clean, fast, and intuitive **Notion-inspired productivity workspace** built with React.

Live Demo: [https://notionflow-two.vercel.app/](https://notionflow-two.vercel.app/)

## ✨ Features

- **Rich Note Taking** with auto-save
- **Powerful Search** (debounced) across notes
- **Kanban Board** for task management
- **Dark Mode** support
- **Fully Responsive** design
- **Persistent Storage** using localStorage
- Clean, modern UI with smooth interactions

## 🛠️ Tech Stack

- **React 18** + Vite
- **Tailwind CSS**
- **Context API** for state management
- **Lucide React** (icons)
- **date-fns** for date formatting
- LocalStorage for persistence
